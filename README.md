# ExperimentedRepositories
Repositories I have previously run. Although coding agents have become more capable, setting up environments and running demos can still take several hours, especially due to dependencies like Conda (which I’m not particularly fond of) and occasionally unpolished codebases. The repositories listed here are ones I have personally tested, so at least something should run out of the box. That said, the setup is somewhat patchwork and not polished enough for full public release. If you have any questions or need help, feel free to reach out.
